## Declaracion de Presentacion

El presente documento describe formalmente el Sistema de Gestion de Activos Fijos como producto de software terminado. La solucion automatiza el ciclo de vida de activos fisicos de una organizacion mediante una plataforma distribuida basada en microservicios, despliegue multi-cloud, frontend web Angular, aplicacion movil React Native, gestion documental, inteligencia artificial, automatizacion de procesos, registro blockchain e inteligencia de negocios.

La documentacion se organiza para evidenciar tres dimensiones de evaluacion: calidad tecnica, productividad del proceso de desarrollo e innovacion aplicada al dominio empresarial.
